# 3 Levels of Money

Dan Koe's framework for understanding your relationship with money and escaping wage slavery.

## Source

Based on: ["The 3 Levels Of Money (How To Escape Wage Slavery)"](https://letters.thedankoe.com/p/the-3-levels-of-money-how-to-escape)

## The Framework

| Level | Name | Percentage | Money As |
|-------|------|------------|----------|
| 1 | Survival | 90% | Master |
| 2 | Success | 9% | Scoreboard |
| 3 | Sacred | 1% | Energy |

## When to Use

- User feels trapped in their job
- User asks about money mindset
- User wants to escape the 9-5
- User feels empty despite making money
- User asks about "life's work"

## Key Concepts

- **You cannot skip levels** - solve current level problems to see next level
- **Development phases**: Dissonance → Uncertainty → Discovery
- **Channels**: Periods of intense development that compress years into months
- **The Meta Model**: Creator path as accessible route to Level 3
