# How to Stand Out in the Dead Internet

Dan Koe's framework for standing out in the AI-saturated internet through signal, taste, and value creation.

## Source

Based on: ["How to stand out in the dead internet"](https://letters.thedankoe.com/p/the-death-of-value-based-content)

## The Core Insight

"Value-based content" isn't dead. Basic educational content is dead. Value has abstracted up a layer into personal narrative, original thought, and taste.

## When to Use

- User asks how to stand out online
- User worries about content saturation
- User asks about AI-generated content competition
- User wants to understand what "value" means now
- User asks about growing on social media
