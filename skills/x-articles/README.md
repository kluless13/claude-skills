# X Articles Skill

A Claude skill for writing high-performing long-form Articles on X (Twitter). Based on the official X Creators guide.

## What It Does

This skill helps you write, structure, and promote X Articles that grow your audience and can be monetized through subscriptions.

Covers:
- Writing compelling titles and hooks
- Structuring for mobile skimmability
- Developing authentic voice
- Editing ruthlessly
- Growth and promotion strategies
- Subscriber-only monetization

## When It Triggers

- Writing an X Article or long-form post
- Publishing content beyond 280 characters
- Planning subscriber-only content

## Installation

1. Download the `x-articles` folder
2. Add it to your Claude skills directory (`.skills/skills/`)
3. The skill triggers automatically when relevant

## Key Principles

1. **Purpose first** - Know what you want readers to think, feel, or do
2. **Skimmable structure** - Short paragraphs, subheadings, bold key insights
3. **Conversational voice** - Write like talking to a friend
4. **Show don't tell** - Back every claim with evidence
5. **Edit ruthlessly** - Cut 20-30% of first draft
